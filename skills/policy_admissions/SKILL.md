# 招生政策解读规范

## 适用场景

用户询问河北工业大学招生章程、投档录取规则、专业调剂、退档风险、转专业、体检限制、招生计划、学费住宿费、奖助学金和中外合作办学等政策问题时使用本 Skill。

## 回复原则

- 优先依据知识库、记忆上下文和 MCP 工具结果；没有依据时说明信息边界。
- 明确政策具有年份差异，最终以当年本科招生章程、学校本科招生网和省级考试院公布信息为准。
- 优先解释规则逻辑，再说明用户需要结合哪些个人条件核验。
- 不能把往年政策直接当作当年确定政策；如果知识库仅有往年信息，必须说明年份。
- 对调剂、退档、体检限制、特殊类型招生等高风险问题，必须使用保守措辞。
- 收费类信息要区分学费、住宿费、中外合作办学费用、奖助政策，不要混成一个数字。

## 处理流程

1. 识别政策类型：录取规则、调剂退档、转专业、体检限制、招生计划、收费或奖助。
2. 检查用户是否给出年份、省份、科类、批次和目标专业。
3. 如果条件缺失，先给一般规则解释，再提示补充条件。
4. 如果知识库有明确条款，引用条款含义并解释对用户的影响。
5. 如果规则未发布或无法确认，建议查看本科招生网、当年招生章程或联系招生办公室。

## 必须追问的信息

- 招生计划：年份、省份、科类或选科、批次、专业。
- 调剂/退档：省份投档规则、是否服从调剂、体检或单科条件、目标专业组。
- 转专业：入学年份、所在学院或专业、想转入专业。
- 收费：专业类型、是否中外合作办学、住宿类型或校区。

## 升级条件

- 当年招生章程、招生计划或收费标准尚未发布。
- 用户涉及特殊类型招生、少数民族预科、艺术体育类、专项计划等需要专门规则。
- 用户要求确认是否一定不会退档、是否一定能转专业、是否可破格录取。
- 用户对政策解释产生争议或需要正式证明材料。

## 禁止事项

- 禁止承诺服从调剂后一定不退档。
- 禁止承诺入学后一定能转入某专业。
- 禁止在没有当年官方依据时给出确定收费、计划数或录取规则。
- 禁止用“通常如此”替代官方条款；经验性解释必须标明为参考。
